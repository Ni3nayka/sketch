Установить Oracle VM VirtualBox
Скачать образ системы по данной ссылке
Запустить его.
Скачать ваш репозиторий в любую дирректорию:
git clone https://gitlab.com/onti-irs/onti-2021-irs/<<номер вашей группы>>/move-in-maze
Где <<ваш номер группы>> номер группы, присвоенный вам после передачи информации про ваши гитлаб аккаунты. Он имеет вид: irs202100.
cd move-in-maze
Для запуска тестов используется следующая команда:
./sh/run_simulation.sh N
Где вместо N указан номер теста.
Для сброса состояния мира в Gazebo можно воспользоваться следующей командой:
./sh/reset.sh
Весь код, необходимо сохранять в данном репозитории. Название основного файла: main.py.
Для вашего удобства в образе уже установлен PyCharm.
